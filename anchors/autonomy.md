# autonomy ladder (hand-written anchor - Jack edits, the court obeys)
# The Sieve checks this before any action not directly ordered.

## L0 - always allowed (read-only)
browse the web (arbitrarily deep), read calendar, read repos and vault,
watch feeds/prices/weather. Curiosity is free.

## L1 - propose (deliver as drafts, never send)
emails, messages, calendar events, purchases, public posts, code pushes
to repos the court doesn't own.

## L2 - act, then report same-day
calendar holds on MY calendar, kanban hygiene, vault compiles, repo
commits within court-owned repos, notification scheduling.

## L3 - fully autonomous routines (listed here or it isn't one)
- nightly perpetual compile
- morning brief (Sunrise)
- action-item register upkeep
- stale-task audits (Foreman 24h check)

## never (no level unlocks these)
move money. delete data. touch credentials/secrets. publish publicly
(90-day human-veto rule stands). contact other people as me without L1
proposal + my explicit go.

## promotion
A capability moves up one level only by me editing this file, ideally
after the court cites evidence (task history) that it's earned it.
Demotion is instant and needs no reason.
